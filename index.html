Dashboard Guide
How to use this dashboard:
  • Select a unit (Noida, Gurugram, or All Units) to filter the data
  • Click "Add Data" to enter new sales records
  • Click edit/pencil icon to modify existing data
  • Click delete/trash icon to remove entries
  • Click Export to Excel to download data as spreadsheet
  • All data is automatically saved in your browser
How to read the charts:
  • Blue bars/lines represent Maersk data
  • Red bars/lines represent Kerry data
  • Green elements show combined totals
  • Purple elements show Noida unit data
  • Orange elements show Gurugram unit data
Dashboard tabs:
  • Combined View: Shows both quantity and value trends
  • Quantity Comparison: Focuses on shipment quantities
  • Value Trend: Shows monetary value progression
